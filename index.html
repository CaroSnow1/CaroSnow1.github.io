Sobre este sitio

Descripción general del proyecto

Se busca generar material audiovisual que le permita la docente poder actualizar o mejorar su material didáctico de clase con el fin de permitirle a la alumna o alumno poder estudiarlo de una forma más practica, cubriendo las necesidades del proceso enseñanza-aprendizaje que las diferentes carreas de ingeniería requieran.

Este sitio web es una propuesta de solución desarrollado bajo el formato del proyecto PAPIME con el cual se pretende identificar y analizar las dificultades que presentan la enseñanza y el aprendizaje de las asignaturas para los profesores y estudiantes de la Facultad de Ingeniería.

Para el desarrollo del sitio se creó material original como videos, infografías, presentaciones y fotografías. Se utilizó el framework de front end Materialize.
